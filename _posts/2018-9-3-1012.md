#### HDU OJ 1012 -- u calculate e

##### 题目说明

$e = \sum_{n}^{i=0} \frac{1}{i!}$

就是给个N，求e的值。这里n默认等于9。

那么问题的点可能就出在最后的输出格式上面吧。

**只有一个提示**：_**数清楚横线的个数= =**_  

---

##### 代码

```c++
#include <iostream>
using namespace std;

int main()
{
	double e = 2.5, before = 0.5;
	cout << "n " << "e" << endl;
	cout << "- -----------" << endl;
	cout << "0 " << "1" << endl;
	cout << "1 " << "2" << endl;
	cout << "2 " << e << endl;
	for (size_t i = 3; i < 10; i++)
	{
		before *= 1.0 / i;
		e += before;
		printf("%d %.9lf\n", i, e);
	}
    return 0;
}
```
